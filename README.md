# 📊 Stock Analysis Template (開源股票分析模板)

一套基於 **波浪理論、黃金分割、均線系統與基本面量化** 的標準化股票分析模板。支援參數化渲染、Python 自動化生成與 GitHub Actions 排程發布。

## ✨ 核心功能
- 📐 **標準化框架**：涵蓋技術面、基本面、情緒面與長中短線策略
- ⚙️ **參數化模板**：使用 Jinja2 語法，一鍵替換變數生成報告
- 🤖 **自動化流水線**：Python 腳本抓取數據 → 計算指標 → 渲染 Markdown
- 🔄 **CI/CD 排程**：透過 GitHub Actions 每日自動生成並推送到 `reports/` 資料夾
- 📜 **開源授權**：MIT License，允許商業/非商業使用與二次開發
